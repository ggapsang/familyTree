1. 엑셀 파일로 정리된 족보(가계도)가 있음
2. 첫 번쨰 시트에는 사람 이름과 생년월일, 성별이 정리되어 있음

| 이름  | 생년월일 | 성별  |
| --- | ---- | --- |
| 이상현 | 1989 | 남   |
| 이윤정 | 1991 | 여   |
| 이민우 | 1961 | 남   |
| 신동화 | 1962 | 여   |
| 이종수 | 1905 | 남   |
| 전일분 | 1907 | 여   |
| 이봉우 | 1951 | 남   |
| 이상진 | 1978 | 남   |

3. 두번쨰 시트에는 부모 자식간의 관계가 정리되어 있음

| 부모  | 자식  |
| --- | --- |
| 이민우 | 이상현 |
| 이민우 | 이윤정 |
| 신동화 | 이상현 |
| 신동화 | 이윤정 |
| 이종수 | 이민우 |
| 이종수 | 이봉우 |
| 이봉우 | 이상진 |

4. 부모의 성별에 따라 자식의 부인지 모인지가 결정됨
5. 같은 자식을 공유하는 부모는 부부가 됨
6. 이런 방식으로 촌수와 가계가 정리될 수 있음

이 데이터를 가지고 트리 맵을 만들어 브라우저에 렌더링 하려고 함

주의사항

- Treant.js 로 구현할 계획
- 유지보수/확장성 고려
- 웹 호스팅 없이 로컬에서 자기 브라우저에서만 나올수 있게
- html 파일 열고, 엑셀 파일 업로드하면, 트리 그려지는 방식으로
- 혹은 해당 폴더 안에 있는 엑셀 파일을 읽어와 트리 그리는 방식으로
