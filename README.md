# 가계도 시각화 프로젝트 (Family Tree Visualization)

엑셀 데이터를 기반으로 가계도를 시각화하는 웹 애플리케이션입니다. 복잡한 가족 관계를 직관적인 트리 구조로 보여주며, 부부 관계와 부모-자식 관계를 명확하게 표현합니다.

## ✨ 주요 기능

*   **엑셀 파일 업로드**: 엑셀 파일(.xlsx, .xls)을 업로드하여 가계도를 생성합니다.
*   **자동 레이아웃**: 가족 관계를 분석하여 자동으로 계층적인 트리 구조를 형성합니다.
*   **부부 관계 표현**: 부부를 나란히 배치하고 붉은 선으로 연결하여 시각적으로 구분합니다.
*   **인터랙티브 뷰**:
    *   **줌/팬 (Zoom/Pan)**: 마우스 휠로 확대/축소하고 드래그하여 이동할 수 있습니다.
    *   **물리 엔진 (Physics)**: 노드를 드래그하면 자연스러운 물리 효과와 함께 재배치됩니다.
*   **데이터 자동 분석**:
    *   이름, 성별, 생년월일 정보를 표시합니다.
    *   직계 가족(Blood)과 인척(In-law) 관계를 시각적으로 구분합니다 (테두리 색상).

## 🛠 기술 스택

*   **HTML5 / CSS3**: 구조 및 스타일링
*   **JavaScript (ES6+)**: 핵심 로직 구현
*   **라이브러리**:
    *   [Cytoscape.js](https://js.cytoscape.org/): 그래프 시각화 엔진
    *   [WebCola](https://ialab.it.monash.edu/webcola/): 물리 기반 레이아웃 엔진
    *   [SheetJS (xlsx)](https://sheetjs.com/): 엑셀 파일 파싱

## 📂 프로젝트 구조

이 프로젝트는 유지보수와 확장성을 위해 **MVC (Model-View-Controller)** 패턴과 유사한 구조로 정리되어 있습니다.

```
FamilyTree/
├── index.html              # 메인 실행 파일
├── css/
│   └── styles.css          # 스타일 시트
├── js/
│   ├── app.js              # 애플리케이션 진입점 (Controller 역할)
│   ├── models/             # 데이터 구조 정의
│   │   ├── person.js       # 사람 데이터 모델
│   │   └── couple.js       # 부부 데이터 모델
│   ├── services/           # 비즈니스 로직
│   │   ├── data-loader.js  # 엑셀 파일 읽기 및 파싱
│   │   └── graph-builder.js # 가계도 알고리즘 (핵심 로직)
│   └── view/               # 화면 렌더링
│       └── renderer.js     # Cytoscape 그래프 렌더링
└── generate_sample_data.js # 테스트용 엑셀 데이터 생성 스크립트
```

## 🚀 사용 방법

1.  **실행**: `index.html` 파일을 웹 브라우저(Chrome, Edge 등)에서 엽니다. (별도의 서버 설치가 필요 없습니다.)
2.  **데이터 준비**: 가계도 데이터가 담긴 엑셀 파일을 준비합니다.
    *   필수 시트: `people` (이름, 성별, 생년월일), `relations` (부모, 자식)
    *   선택 시트: `couples` (이름, 배우자) - 없어도 부모-자식 관계를 통해 자동 추론됩니다.
3.  **업로드**: 화면의 "파일 선택" 버튼을 눌러 엑셀 파일을 업로드합니다.
4.  **확인**: 화면에 가계도가 그려집니다. 마우스로 자유롭게 탐색하세요.
